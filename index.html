1.1 การเริ่มต้นเขียนโปรแกรม

การเขียนโปรแกรม (Programming) คือกระบวนการออกแบบและสร้างชุดคำสั่งให้คอมพิวเตอร์ทำงานตามที่เราต้องการ

กระบวนการพื้นฐานที่ควรคุ้นเคย ประกอบด้วย 4 ขั้นตอน:

วิเคราะห์ปัญหา → ออกแบบอัลกอริทึม → เขียนโค้ด → ทดสอบและแก้ไข

นิยาม

โปรแกรม (Program) คือลำดับคำสั่งที่คอมพิวเตอร์สามารถประมวลผลได้ โดยเริ่มจากจุดเข้า main() และทำงานทีละคำสั่ง

ขั้นตอนการพัฒนาโปรแกรม

  1. เขียน source code ในไฟล์ .cpp เช่น main.cpp
  2. คอมไพล์ด้วย g++ เพื่อสร้างไฟล์ executable
  3. รันโปรแกรมและตรวจผลลัพธ์
  4. แก้ไข (debug) จนได้ผลถูกต้องตามโจทย์
เพิ่มเติม

ในการแข่งขัน ให้เริ่มจากอ่านโจทย์ให้ครบ ระบุ input/output format แล้วเขียนโค้ดทีละส่วน อย่ารีบส่งก่อนทดสอบกับตัวอย่างในโจทย์

1.2 การใช้งานคอมไพล์เลอร์ g++

g++ เป็นคอมไพล์เลอร์ GNU สำหรับ C++ ที่ใช้กันแพร่หลายในระบบจัดการการแข่งขัน (เช่น DOMjudge, CMS) บน Linux, macOS และ Windows (MinGW/MSYS2)

คำสั่งคอมไพล์มาตรฐาน

g++ -std=c++20 -Wall -Wextra -O2 -o program main.cpp
./program

คำสั่งใน Terminal / Command Prompt

ตัวเลือก ความหมาย
-std=c++20 ใช้มาตรฐาน C++20
-Wall -Wextra แสดงคำเตือน
-O2 เปิดการ optimize
-o ชื่อ ตั้งชื่อไฟล์ output

สรุป flag ที่ใช้บ่อย

คอมไพล์หลายไฟล์

g++ -std=c++20 -Wall -o program main.cpp utils.cpp math.cpp
หมายเหตุ

ถ้า compile ไม่ผ่าน ให้อ่าน error จากบรรทัดแรกที่เกี่ยวข้อง มักบอกเลขบรรทัดและสาเหตุ เช่น ลืม semicolon, วงเล็บไม่ครบ

1.3 เครื่องมือในการเขียนโปรแกรม (IDE)

IDE (Integrated Development Environment) รวม editor, debugger, build system และบางครั้งมี version control ไว้ในที่เดียว ช่วยให้เขียนและ debug ได้เร็วขึ้น

Code::Blocks / Dev-C++ ติดตั้งง่าย ผู้เริ่มต้น

ให้เรียนรู้ทั้ง IDE และคำสั่ง g++ ใน terminal เพราะในการสอบและการแข่งขัน มักต้อง compile บน server โดยไม่มี IDE

1.4 การลงรหัส (Coding)

การลงรหัส (Coding) คือการแปลงอัลกอริทึมที่ออกแบบไว้ ให้เป็นคำสั่ง C++ ที่คอมพิวเตอร์เข้าใจ โค้ดที่อ่านง่ายช่วยให้ตรวจงาน ทำงานเป็นทีม และ debug ได้เร็วขึ้น

หลักการเขียนโค้ดที่ดี

  • ตั้งชื่อตัวแปร/ฟังก์ชันให้สื่อความหมาย เช่น studentCount แทน x
  • จัด indent ให้สม่ำเสมอ (แนะนำ 4 spaces)
  • แบ่งปัญหาใหญ่เป็นฟังก์ชันย่อย ๆ
  • เขียน comment อธิบาย ทำไม ไม่ใช่แค่ ทำอะไร
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl; // แสดงข้อความ
    return 0;  // โปรแกรมทำงานสำเร็จ
}

โปรแกรม Hello World แบบอ่านง่าย — มี comment และ return 0

Template

เก็บ template โปรแกรมมาตรฐานไว้ copy ใช้ทุกครั้ง จะประหยัดเวลาในการแข่งขัน

#include <bits/stdc++.h>

    ios_base::sync_with_stdio(false);

    // your code here

}

1.5 การรับข้อมูลและแสดงผลเบื้องต้น

#include <iostream>

คอมพิวเตอร์ประมวลผลด้วยคำสั่งไบนารี (0 และ 1) ที่ CPU ทำความเข้าใจโดยตรง มนุษย์ไม่สามารถเขียน machine code ได้อย่างมีประสิทธิภาพ จึงต้องใช้ภาษาระดับสูง เช่น C++ เป็นตัวกลาง

คอมไพล์เลอร์แปลง source code เป็น object code แล้ว linker รวมเป็น executable ที่รันได้

2.2 ภาษา C++ ในการเขียนโปรแกรม

C++ พัฒนาต่อจาก C โดย Bjarne Stroustrup ปี 1983 รองรับ procedural, object-oriented และ generic programming ใช้กันแพร่หลายในอุตสาหกรรมซอฟต์แวร์ และการเขียนโปรแกรมเชิงแข่งขัน

  • ประสิทธิภาพสูง — ใกล้ hardware เหมาะกับงานที่ต้องเร็ว
  • STL — มี container และ algorithm สำเร็จรูป
  • หลาย paradigm — procedural, OOP, template
  • ข้ามแพลตฟอร์ม — Linux, Windows, macOS

C++ เป็นภาษาหลักใน IOI, TOI และ ICPC เพราะ STL ช่วยลดเวลา implement

2.3 โครงสร้างของโปรแกรม C++

โปรแกรม C++ ประกอบด้วยส่วนหลัก: preprocessor directives, declarations, definitions และฟังก์ชัน main 

#include <iostream>    // 1) นำเข้า library (preprocessor)
using namespace std;   // 2) ใช้ namespace (optional)

int add(int a, int b) { // 3) ฟังก์ชันช่วย
    return a + b;
}

int main() {            // 4) จุดเริ่มต้นโปรแกรม
    int result = add(3, 4);
    cout << result << endl;
    return 0;
}

หมายเหตุ

2.4 ฟังก์ชัน main และ namespace

main() เป็นฟังก์ชันแรกที่ OS เรียกเมื่อรันโปรแกรม

namespace จัดกลุ่มชื่อป้องกันการชนกัน — std คือ namespace ของ Standard Library

#include <iostream>

namespace Math {
    int add(int a, int b) { return a + b; }
    int mul(int a, int b) { return a * b; }
}

int main() {
    std::cout << Math::add(3, 4) << std::endl;  // 7
    std::cout << Math::mul(3, 4) << std::endl;  // 12
    return 0;
}

namespace แยกกลุ่มฟังก์ชัน

3.1 ชนิดของข้อมูล (Data Types)

ชนิดข้อมูล (data type) กำหนดว่าตัวแปรเก็บค่าประเภทใด ใช้หน่วยความจำเท่าใด และทำ operation อะไรได้

ชนิด ขนาด ช่วงค่า สำหรับ
int 4 bytes -2,147,483,648 ถึง 2,147,483,647 จำนวนเต็ม
long long 8 bytes -9,223,372,036,854,775,808 ถึง 9,223,372,036,854,775,807 จำนวนเต็ม
double 8 bytes 1.7 × 10⁻³⁰⁸ ถึง 1.7 × 10³⁰⁸ ทศนิยม
char 1 byte -128 ถึง 127 ตัวอักษร
bool 1 byte true/false เงื่อนไข

ชนิดข้อมูลพื้นฐานใน C++

ข้อควรระวัง

ถ้าคูณหรือบวกแล้วมากกว่า 2 × 10⁹ หรือน้อยกว่า -2 × 10⁹ ให้ใช้ long long

#include <iostream>
using namespace std;

int main() {
    int count = 100;
    long long bigNumber = 1000000000000LL;
    double pi = 3.14159;
    char grade = 'A';
    bool passed = true;

    cout << count << " " << bigNumber << " " << pi << endl;
    return 0;
}

Output

3.2 ตัวแปร (Variables)

ตัวแปร (variable) เป็นชื่อที่อ้างถึงพื้นที่ในหน่วยความจำ ต้องประกาศชนิดก่อนใช้งาน 

#include <iostream>
using namespace std;

int main() {
    int score = 85;       // ประกาศ + กำหนดค่าเริ่มต้น
    int bonus;            // ประกาศอย่างเดียว (ค่าไม่แน่นอน!)
    bonus = 10;           // กำหนดค่าทีหลัง

    score = score + bonus;  // เปลี่ยนค่าได้
    cout << score << endl;  // 95
    return 0;
}

การประกาศและเปลี่ยนค่าตัวแปร

ข้อควรระวัง

ตัวแปรที่ประกาศแต่ไม่ initialize อาจมีค่าขยะ (garbage value) — ควรกำหนดค่าเสมอ

9.1 แนวคิดของ Recursion

9.2 Base Case และ Recursive Case

#include <iostream>
using namespace std;

int sum(int n) {
    if (n <= 0) return 0;        // base case: หยุด
    return n + sum(n - 1);       // recursive case
}

    cout << sum(5) << endl;  // 15 (= 5+4+3+2+1)
    return 0;
}

12.2 Bubble Sort, Selection Sort และ Insertion Sort

Bubble Sort — O(n²)

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
            }
        }
    }
    return ;
}

Bubble Sort

Bubble Sort สลับคู่ที่ adjacent จนเรียงครบ — เข้าใจง่าย แต่ช้า

Selection Sort — O(n²)

void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {      
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx]) {
                min_idx = j;
            }
        }
        int temp = arr[i];
        arr[i] = arr[min_idx];
        arr[min_idx] = temp;
    }
    return ;
}

Selection Sort

Selection Sort เลือกค่าน้อยที่สุดจากส่วนที่ยังไม่เรียง แล้วสลับมาไว้ด้านหน้า — เข้าใจง่าย แต่ช้า

Insertion Sort — O(n²)

void insertionSort(int arr[], int n)
{
    for (int i = 1; i < n; ++i) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }
        arr[j + 1] = key;
    }
    return ;
}

Insertion Sort

Insertion Sort แทรกสมาชิกแต่ละตัวเข้าไปในตำแหน่งที่ถูกต้องของส่วนที่เรียงแล้ว — เข้าใจง่าย และมีประสิทธิภาพเมื่อข้อมูลเกือบเรียงอยู่แล้ว

12.4 การค้นหาแบบ Binary Search

#include <iostream>
using namespace std;

int binarySearch(int arr[], int n, int target) {
    int lo = 0, hi = n - 1;
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;  // ป้องกัน overflow
        if (arr[mid] == target) return mid;
        if (arr[mid] < target) lo = mid + 1;
        else                   hi = mid - 1;
    }
    return -1;
}

int main() {
    int arr[] = {1, 3, 5, 7, 9, 11};
    cout << binarySearch(arr, 6, 7) << endl;  // 3
    return 0;
}

14.1 แนวคิดของ Linked List

Linked List เป็นโครงสร้างข้อมูลเชิงเส้นที่เก็บข้อมูลเป็นชุดของ node แต่ละ node จะเชื่อมต่อกับ node ถัดไปด้วย pointer หรือ reference ทำให้ข้อมูลไม่จำเป็นต้องจัดเก็บแบบต่อเนื่องในหน่วยความจำ

14.2 Singly Linked List

Singly Linked List เป็น linked list แบบพื้นฐานที่สุด แต่ละ node มี pointer ชี้เฉพาะไปยัง node ถัดไป ทำให้เดินทางจากหัวไปหาหางได้ง่าย แต่ย้อนกลับไม่ได้โดยตรง

ตัวอย่างฟังก์ชันพื้นฐานที่ควรรู้:

  • pushFront — แทรก node ที่ต้นรายการ
  • pushBack — แทรก node ที่ท้ายรายการ
  • insertAfter — แทรก node หลัง node ที่กำหนด
  • deleteNode — ลบ node ที่ต้องการ
  • printList — แสดงรายการจากหัวถึงท้าย
#include <iostream>
using namespace std;

struct Node {
    int data;      // ค่าที่เก็บใน node
    Node* next;    // pointer ไปยัง node ถัดไป
};

Node* createNode(int value) {
    return new Node{value, nullptr};
}

void pushFront(Node*& head, int value) {
    Node* newNode = createNode(value);
    newNode->next = head;
    head = newNode;
}

void pushBack(Node*& head, int value) {
    Node* newNode = createNode(value);
    if (head == nullptr) {
        head = newNode;
        return;
    }
    Node* p = head;
    while (p->next != nullptr) {
        p = p->next;
    }
    p->next = newNode;
}

void insertAfter(Node* prev, int value) {
    if (prev == nullptr) return;
    Node* newNode = createNode(value);
    newNode->next = prev->next;
    prev->next = newNode;
}

void deleteNode(Node*& head, int value) {
    if (head == nullptr) return;
    while (head != nullptr && head->data == value) {
        Node* temp = head;
        head = head->next;
        delete temp;
    }
    Node* p = head;
    while (p != nullptr && p->next != nullptr) {
        if (p->next->data == value) {
            Node* temp = p->next;
            p->next = p->next->next;
            delete temp;
        } else {
            p = p->next;
        }
    }
}

void printList(Node* head) {
    for (Node* p = head; p != nullptr; p = p->next) {
        cout << p->data;
        if (p->next != nullptr) cout << " -> ";
    }
    cout << "\n";
}

int main() {
    Node* head = nullptr;
    pushBack(head, 10);
    pushBack(head, 20);
    pushFront(head, 5);
    insertAfter(head->next, 15);  // แทรกหลัง node ที่มีค่า 10
    printList(head);              // 5 -> 10 -> 15 -> 20
    deleteNode(head, 10);
    printList(head);              // 5 -> 15 -> 20
    return 0;
}

Singly Linked List — รวมฟังก์ชันพื้นฐาน

  • pushFront และ pushBack เป็น 2 วิธีหลักในการเพิ่ม node
  • insertAfter ช่วยให้แทรก node ได้ตรงตำแหน่งที่ต้องการ
  • deleteNode ต้องอัปเดต pointer ก่อน delete ทุกครั้ง

14.3 Doubly Linked List

Doubly Linked List เป็น linked list ที่ node แต่ละตัวเก็บ pointer สองตัว: ชี้ไปยัง node ก่อนหน้า (prev) และ node ถัดไป (next)

โครงสร้างนี้ช่วยให้เดินทางทั้งไปข้างหน้าและย้อนกลับได้สะดวกกว่า singly linked list และยังทำให้การลบ node ตรงกลางง่ายขึ้น

ฟังก์ชันหลักที่ควรรู้:

  • pushFront — เพิ่ม node ที่ต้นรายการ
  • pushBack — เพิ่ม node ที่ท้ายรายการ
  • insertAfter — แทรก node หลัง node ที่กำหนด
  • deleteNode — ลบ node ที่กำหนดโดยไม่ต้องค้นจากหัวใหม่
  • printForward — แสดงรายการจากหัวไปท้าย
  • printBackward — แสดงรายการจากท้ายไปหัว
#include <iostream>
using namespace std;

struct DNode {
    int data;
    DNode* prev;
    DNode* next;
};

DNode* createDNode(int value) {
    return new DNode{value, nullptr, nullptr};
}

void pushFront(DNode*& head, int value) {
    DNode* node = createDNode(value);
    node->next = head;
    if (head != nullptr) head->prev = node;
    head = node;
}

void pushBack(DNode*& head, int value) {
    DNode* node = createDNode(value);
    if (head == nullptr) {
        head = node;
        return;
    }
    DNode* p = head;
    while (p->next != nullptr) p = p->next;
    p->next = node;
    node->prev = p;
}

void insertAfter(DNode* prev, int value) {
    if (prev == nullptr) return;
    DNode* node = createDNode(value);
    node->next = prev->next;
    node->prev = prev;
    prev->next = node;
    if (node->next != nullptr) node->next->prev = node;
}

void deleteNode(DNode*& head, DNode* target) {
    if (head == nullptr || target == nullptr) return;
    if (head == target) {
        head = head->next;
        if (head != nullptr) head->prev = nullptr;
        delete target;
        return;
    }
    if (target->prev != nullptr) target->prev->next = target->next;
    if (target->next != nullptr) target->next->prev = target->prev;
    delete target;
}

void printForward(DNode* head) {
    for (DNode* p = head; p != nullptr; p = p->next) {
        cout << p->data;
        if (p->next != nullptr) cout << " <-> ";
    }
    cout << "\n";
}

void printBackward(DNode* tail) {
    for (DNode* p = tail; p != nullptr; p = p->prev) {
        cout << p->data;
        if (p->prev != nullptr) cout << " <-> ";
    }
    cout << "\n";
}

int main() {
    DNode* head = nullptr;
    pushBack(head, 10);
    pushBack(head, 20);
    pushFront(head, 5);
    insertAfter(head, 15);    // แทรกหลัง node แรก
    printForward(head);       // 5 <-> 15 <-> 10 <-> 20

    DNode* tail = head;
    while (tail->next != nullptr) tail = tail->next;
    printBackward(tail);      // 20 <-> 10 <-> 15 <-> 5

    deleteNode(head, head->next);  // ลบ node ที่มีค่า 15
    printForward(head);            // 5 <-> 10 <-> 20
    return 0;
}

Doubly Linked List — ฟังก์ชันเพิ่ม, ลบ, และพิมพ์ข้อมูล

  • pushFront เพิ่ม node ที่ต้นรายการ โดยอัปเดต prev ของ node เดิม
  • pushBack ต้องเดินหา node สุดท้ายแล้วเชื่อม node ใหม่ที่ท้าย
  • insertAfter เพิ่ม node หลัง node ที่กำหนดและเชื่อม pointer สองฝั่งให้ถูกต้อง
  • deleteNode ลบ node โดยอาศัย prev และ next ของ target
  • printBackward แสดงผลย้อนกลับจากท้ายไปหัว ใช้ง่ายเมื่อมี prev

14.4 Circular Linked List

Circular Linked List เป็น linked list ที่ node สุดท้ายจะชี้กลับไปยัง node แรก แทนที่จะชี้ไปยัง nullptr ทำให้โครงสร้างกลายเป็นวงกลม

โครงสร้างแบบนี้เหมาะกับการทำงานที่ต้องวนรอบข้อมูลเป็นลูป เช่น playlist, scheduler, หรือระบบรอบซ้ำ

ฟังก์ชันสำคัญที่ควรรู้:

  • pushBack — เพิ่ม node ใหม่ที่ท้ายแล้วอัปเดต tail
  • insertAfter — แทรก node หลัง node ที่กำหนดในลูป
  • deleteValue — ลบ node ตามค่าและปรับ tail หากจำเป็น
  • printCircular — แสดงรายการโดยวนกลับจนถึงหัว
#include <iostream>
using namespace std;

struct CNode {
    int data;
    CNode* next;
};

CNode* createCNode(int value) {
    return new CNode{value, nullptr};
}

void pushBack(CNode*& tail, int value) {
    CNode* node = createCNode(value);
    if (tail == nullptr) {
        node->next = node;
        tail = node;
        return;
    }
    node->next = tail->next;
    tail->next = node;
    tail = node;
}

void insertAfter(CNode* prev, int value) {
    if (prev == nullptr) return;
    CNode* node = createCNode(value);
    node->next = prev->next;
    prev->next = node;
}

void deleteValue(CNode*& tail, int value) {
    if (tail == nullptr) return;
    CNode* curr = tail->next;
    CNode* prev = tail;
    bool found = false;
    do {
        if (curr->data == value) {
            found = true;
            prev->next = curr->next;
            if (curr == tail) {
                tail = (curr == prev) ? nullptr : prev;
            }
            delete curr;
            break;
        }
        prev = curr;
        curr = curr->next;
    } while (curr != tail->next);
}

void printCircular(CNode* tail) {
    if (tail == nullptr) return;
    CNode* head = tail->next;
    CNode* p = head;
    do {
        cout << p->data;
        p = p->next;
        if (p != head) cout << " -> ";
    } while (p != head);
    cout << "\n";
}

int main() {
    CNode* tail = nullptr;
    pushBack(tail, 10);
    pushBack(tail, 20);
    pushBack(tail, 30);
    insertAfter(tail->next, 15); // แทรกหลัง node แรก
    printCircular(tail);         // 10 -> 15 -> 20 -> 30
    deleteValue(tail, 20);
    printCircular(tail);         // 10 -> 15 -> 30
    return 0;
}

Circular Linked List — ฟังก์ชันพื้นฐานสำหรับการเพิ่ม, แทรก, ลบ, และพิมพ์

  • pushBack เพิ่ม node ใหม่ที่ท้ายโดยไม่ตัดลูป
  • insertAfter เพิ่ม node ใหม่สำหรับตำแหน่งใด ๆ ในวงกลม
  • deleteValue ต้องตรวจสอบกรณีลบ node แรก/สุดท้ายและปรับ tail ให้ถูกต้อง

14.5 ตัวอย่างการใช้งาน Linked List

Linked list ถูกใช้ในหลายงานที่ต้องการโครงสร้างข้อมูลแบบยืดหยุ่น เช่น ในการจัดการหน่วยความจำแบบ dynamic, สร้าง adjacency list สำหรับกราฟ, หรือใช้เป็นฐานของ stack และ queue

ตัวอย่างการใช้งานที่พบบ่อยมีดังนี้:

  • เก็บรายการเพลงหรือ playlist ที่สามารถเพิ่ม/ลบเพลงได้บ่อย
  • เก็บโหนดกราฟในรูปแบบ adjacency list เพื่อประหยัดพื้นที่เมื่อกราฟมีค่าเป็น sparse
  • ใช้งานใน text editor สำหรับ undo/redo เพราะสามารถเพิ่มและลบข้อมูลได้ง่าย
  • ใช้สร้าง stack หรือ queue เมื่อไม่ต้องการกำหนดขนาดล่วงหน้า
void append(Node*& head, int value) {
    Node* newNode = new Node{value, nullptr};
    if (head == nullptr) {
        head = newNode;
        return;
    }
    Node* p = head;
    while (p->next != nullptr) p = p->next;
    p->next = newNode;
}

void removeValue(Node*& head, int value) {
    while (head != nullptr && head->data == value) {
        Node* temp = head;
        head = head->next;
        delete temp;
    }
    Node* p = head;
    while (p != nullptr && p->next != nullptr) {
        if (p->next->data == value) {
            Node* temp = p->next;
            p->next = p->next->next;
            delete temp;
        } else {
            p = p->next;
        }
    }
}

ตัวอย่างเพิ่มเติม: เพิ่มท้ายและลบค่าที่ต้องการ

เพิ่มเติม

ถ้าต้องการเข้าถึงตำแหน่งบ่อย ๆ หรือเก็บข้อมูลขนาดคงที่ array/vector จะทำได้ดีกว่า แต่ linked list ดีเมื่อต้องแก้ไขตำแหน่งบ่อย

15.1 แนวคิดของ Stack (LIFO)

Stack เป็นโครงสร้างข้อมูลแบบ LIFO (Last In, First Out) — ข้อมูลที่ถูกใส่เข้ามาล่าสุดจะถูกเอาออกก่อนเสมอ การใช้งานจริงมักจะใส่หรือหยิบข้อมูลเฉพาะจากด้านบน (top) เท่านั้น

นิยาม

Stack — โครงสร้างที่รองรับ push (ใส่), pop (เอาออก), top (ดูตัวบนสุด)

ตัวอย่างในชีวิตจริง: จานซ้อนกัน — ใส่หรือหยิบได้เฉพาะจานด้านบนสุด

  • Stack ช่วยให้ย้อนกลับสถานะได้ง่าย เช่น undo/redo
  • ใช้เก็บข้อมูลระหว่างการทำ recursion และ DFS
  • ใช้ตรวจสอบวงเล็บสมดุลและประมวลผลนิพจน์คณิตศาสตร์

15.2 การ Implement Stack ด้วยอาร์เรย์

#include <iostream>
using namespace std;

const int MAXN = 1000;
int stackArr[MAXN];
int topIndex = -1;

void push(int value) {
    stackArr[++topIndex] = value;
}

int pop() {
    return stackArr[topIndex--];
}

int peek() {
    return stackArr[topIndex];
}

bool isEmpty() {
    return topIndex < 0;
}

int main() {
    push(10);
    push(20);
    push(30);
    cout << peek() << endl;  // 30
    cout << pop() << endl;   // 30
    cout << pop() << endl;   // 20
    return 0;
}

คำเตือน

สแตกแบบ array มีขนาดคงที่ หากใส่เกินขนาดจะเกิด overflow และต้องเช็คให้ดีว่าไม่ดึงออกจาก stack ว่าง

15.3 การ Implement Stack ด้วย Linked List

struct StackNode {
    int data;
    StackNode* next;
};

void pushNode(StackNode*& top, int value) {
    StackNode* node = new StackNode{value, top};
    top = node;
}

int popNode(StackNode*& top) {
    int value = top->data;
    StackNode* temp = top;
    top = top->next;
    delete temp;
    return value;
}

เพิ่มเติม

Stack ด้วย linked list ทำให้ push และ pop เป็น O(1) ตราบใดที่ชี้ top อยู่เสมอ

15.4 std::stack ใน STL

#include <iostream>
#include <stack>
#include <string>
using namespace std;

bool isBalanced(string s) {
    stack<char> st;
    for (char c : s) {
        if (c == '(' || c == '[' || c == '{')
            st.push(c);
        else if (c == ')' || c == ']' || c == '}') {
            if (st.empty()) return false;
            st.pop();
        }
    }
    return st.empty();
}

int main() {
    cout << isBalanced("({[]})") << endl;  // 1 (true)
    cout << isBalanced("([)]") << endl;    // 0 (false)
    return 0;
}

16.1 แนวคิดของ Queue (FIFO)

    queueArr[++rearIndex] = value;
}

int dequeue() {
    return queueArr[frontIndex++];

    return frontIndex > rearIndex;

16.3 การ Implement Queue ด้วย Linked List

struct QueueNode {
    int data;
    QueueNode* next;
};

struct Queue {
    QueueNode *front = nullptr, *rear = nullptr;

    void enqueue(int value) {
        QueueNode* node = new QueueNode{value, nullptr};
        if (rear == nullptr) {
            front = rear = node;
        } else {
            rear->next = node;
            rear = node;
        }
    }

    int dequeue() {
        int value = front->data;
        QueueNode* temp = front;
        front = front->next;
        if (front == nullptr) rear = nullptr;
        delete temp;
        return value;
    }
};
